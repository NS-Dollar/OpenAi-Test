# Nico Schmitz Website

Dieses Repository enthält eine einfache Angular-Anwendung mit Node.js-Backend.

## Entwicklung

1. Abhängigkeiten installieren:
   ```bash
   npm install --prefix frontend
   npm install --prefix backend
   ```
2. Backend starten:
   ```bash
   npm start --prefix backend
   ```
3. Frontend starten:
   ```bash
   npm start --prefix frontend
   ```
Die Angular-App läuft standardmäßig auf http://localhost:4200 und das Backend auf Port 3000.
